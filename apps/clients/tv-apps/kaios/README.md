# KaiOS App - StreamVerse

KaiOS (Firefox OS heritage) application for StreamVerse.

## Overview

Web-based mobile app optimized for KaiOS feature phones with TV connectivity.

## Features

- ✅ Web-based UI
- ✅ Mobile-optimized interface
- ✅ Video playback support
- ✅ Basic DRM via EME

## Building

Package as web app for KaiStore.
