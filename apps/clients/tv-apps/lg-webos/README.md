# LG webOS TV App - StreamVerse

Native LG webOS TV application for StreamVerse platform.

## Overview

This webOS TV app provides a native TV experience using web technologies with Magic Remote support.

## Features

- ✅ Authentication with token management
- ✅ Content browsing with horizontal rows
- ✅ Video playback with webOS MediaPlayer
- ✅ DRM support (PlayReady, Widevine via EME)
- ✅ Magic Remote navigation
- ✅ Search functionality
- ✅ Settings screen

## Prerequisites

- LG webOS SDK
- webOS TV Emulator or real device
- LG Developer Account

## Building

```bash
# Package for webOS
ares-package .

# Output: StreamVerse_1.0.0_all.ipk
```

## Installation

```bash
# Install on device/emulator
ares-install StreamVerse_1.0.0_all.ipk
```
