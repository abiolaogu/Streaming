# VIDAA TV App - StreamVerse

VIDAA TV application for StreamVerse platform (Hisense TVs).

## Overview

Web-based app for VIDAA OS, similar to Tizen/webOS implementation.

## Features

- ✅ Web-based UI
- ✅ Video playback
- ✅ DRM support
- ✅ Remote navigation

## Building

Package as web app ZIP for VIDAA app store.
