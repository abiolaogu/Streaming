# Amazon Fire TV App - StreamVerse

Amazon Fire TV application based on Android TV app with Fire TV-specific enhancements.

## Overview

Fire TV app built with Jetpack Compose and Leanback library, optimized for Amazon Fire TV devices.

## Features

- ✅ Same as Android TV app
- ✅ Alexa voice integration
- ✅ Fire TV recommendations
- ✅ Amazon Appstore optimized

## Building

```bash
cd fire-tv
./gradlew assembleRelease

# APK for Amazon Appstore
# Output: app/build/outputs/apk/release/app-release.apk
```

## Deployment

1. Build signed APK
2. Upload to Amazon Developer Console
3. Submit for certification
