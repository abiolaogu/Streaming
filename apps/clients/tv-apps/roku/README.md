# Roku TV App - StreamVerse

Roku BrightScript application for StreamVerse platform.

## Overview

Native Roku channel built with BrightScript and SceneGraph for optimal TV experience.

## Features

- ✅ SceneGraph UI
- ✅ Content browsing
- ✅ Video playback (roVideoPlayer)
- ✅ DRM support
- ✅ Remote control navigation
- ✅ Search functionality

## Prerequisites

- Roku Developer Account
- Roku Developer SDK
- Roku device or Roku TV

## Building

1. Package channel in Roku Developer Dashboard
2. Or use `roku-deploy` tool
3. Upload to Roku Channel Store

## Testing

```bash
# Side-load on device
roku-deploy . --host <device-ip>
```
