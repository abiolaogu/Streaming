## Creating Your Account

### Step 1: Sign Up

1. Visit [streamverse.io](https://streamverse.io)
2. Click **"Sign Up"**
3. Enter your email and create a password
4. Choose a subscription plan:
   - **Basic** ($9.99/month) - SD quality, 1 screen
   - **Standard** ($14.99/month) - HD quality, 2 screens
   - **Premium** ($19.99/month) - 4K quality, 4 screens

5. Enter payment information
6. Click **"Start Watching"**

### Step 2: Email Verification

1. Check your email for verification link
2. Click the link to verify your account
3. You're ready to stream!

### Alternative Sign-Up Methods

- **Google**: Sign up with your Google account
- **Apple**: Sign up with Apple ID (iOS devices)
- **Facebook**: Sign up with Facebook account

## Setting Up Profiles

### Creating a Profile

1. Click your **profile icon** (top right)
2. Select **"Manage Profiles"**
3. Click **"Add Profile"**
4. Enter profile name
5. Choose an avatar
6. Select profile type:
   - **Adult Profile**: Full access
   - **Kids Profile**: Age-appropriate content only
7. Click **"Save"**

### Customizing Your Profile

- **Avatar**: Choose from 50+ avatars or upload your own
- **Language**: Set preferred language for interface and content
- **Maturity Rating**: Set content rating limits (G, PG, PG-13, R, NC-17)
- **Autoplay**: Enable/disable auto-play for next episode

### Switching Profiles

1. Click profile icon
2. Select the profile you want to use
3. Start watching!

## Troubleshooting

### Video Won't Play

**Solutions:**
1. Check internet connection (5 Mbps minimum)
2. Refresh the page
3. Clear browser cache
4. Update browser to latest version
5. Disable VPN/proxy
6. Try different browser

### Buffering Issues

**Solutions:**
1. Check connection speed at [fast.com](https://fast.com)
2. Lower video quality
3. Close other apps using internet
4. Restart router
5. Move closer to Wi-Fi router
6. Use wired connection if possible

### Audio/Video Out of Sync

**Solutions:**
1. Pause and resume playback
2. Refresh page
3. Clear cache
4. Update app/browser
5. Check for device software updates

### Subtitles Not Showing

**Solutions:**
1. Click CC icon to enable
2. Select correct language
3. Refresh player
4. Check if content has subtitles available

### Can't Sign In

**Solutions:**
1. Verify email and password
2. Use "Forgot Password" to reset
3. Check for typos in email
4. Clear browser cookies
5. Try different browser
6. Contact support

### Download Failures

**Solutions:**
1. Check storage space
2. Verify stable Wi-Fi connection
3. Update app to latest version
4. Re-download content
5. Restart device

### Picture Quality Poor

**Solutions:**
1. Check internet speed (25 Mbps for 4K)
2. Manually select higher quality
3. Verify subscription plan supports quality
4. Check HDMI cable (for TV)
5. Adjust TV picture settings

### Casting Not Working

**Solutions:**
1. Ensure devices on same Wi-Fi network
2. Restart Chromecast/Apple TV
3. Update streaming device firmware
4. Restart app
5. Reboot router

## Glossary

**4K/Ultra HD**: Highest quality video (3840x2160 pixels)

**HDR**: High Dynamic Range - enhanced color and brightness

**Streaming**: Watching video over internet without downloading

**Download**: Save content to device for offline viewing

**Watchlist**: Personal collection of saved titles

**Profile**: Individual account within main subscription

**Subtitles**: Text display of dialogue

**Audio Description**: Narration describing visual elements for visually impaired

**Parental Controls**: Restrictions to limit content access

**Maturity Rating**: Content age-appropriateness rating (G, PG, PG-13, R, NC-17)
