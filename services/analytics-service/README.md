# Analytics Service

Microservice for data collection and processing (Python with FastAPI).

## Features

- ✅ Event ingestion (REST API, Kafka consumer)
- ✅ Metrics calculation (views, watch time, completion rate)
- ✅ Data storage (MongoDB, ClickHouse)
- ✅ Reporting API

## Running

```bash
pip install -r requirements.txt
python main.py
```
