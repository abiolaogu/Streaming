# Notification Service

Multi-channel notification service (Node.js with Express).

## Features

- ✅ Push notifications (FCM, Web Push)
- ✅ Email notifications (SendGrid, AWS SES)
- ✅ SMS notifications (Twilio, Africa Talking)
- ✅ In-app notifications
- ✅ Queue management (RabbitMQ)

## Running

```bash
npm install
npm start
```
